Principais elementos do Markdown:
Títulos:
# Título 1
## Título 2
### Título 3

Negrito e itálico:
**negrito**
*itálico*

Listas:
- Item de lista
1. Item numerado

Links:
[texto do link](url)

Imagens:
![texto alternativo](url-da-imagem)

Código:
`código` para inline

<pre> ``` bloco de código ``` </pre>
Citações:
> citação
